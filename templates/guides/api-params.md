## 예시 JSON

```json
{
  "prompt": "your detailed prompt here...",
  "aspectRatio": "9:16",
  "duration": "8s",
  "resolution": "1080p"
}
```

## 파라미터 의미
- `prompt`: 상세 프롬프트 텍스트
- `aspectRatio`: 화면 비율
- `duration`: 영상 길이
- `resolution`: 출력 해상도
