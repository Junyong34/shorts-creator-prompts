## 권장 길이
- 80-150 words

## 구조 템플릿

```
[ASPECT RATIO & FORMAT]
9:16 vertical video format, portrait mode, optimized for mobile screen.

[VISUAL STYLE]
{스타일: cinematic / realistic / animated / hyper-realistic / surreal}

[SCENE & ACTION]
{메인 장면 설명 + 주요 동작}

[SUBJECT]
{피사체 상세 설명 - 중앙 배치 권장}

[ENVIRONMENT]
{배경, 장소, 시간대, 시대}

[CAMERA]
{카메라 무빙: static / tilt up / tilt down / dolly in / close-up / crane shot}

[LIGHTING & MOOD]
{조명: soft golden-hour / high contrast / neon-lit / dramatic shadows}
{분위기: dreamy / energetic / mysterious / warm}

[AUDIO] (Optional)
{배경음악, 환경음, 효과음}
```
