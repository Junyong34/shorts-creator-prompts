

# 🎬 AI 애니메이션 제작 워크플로우 템플릿

유튜브 AI 애니메이션 제작을 위한 단계별 워크플로우 정리 문서입니다.

---

## 📌 1단계: 캐릭터 페르소나 및 이미지 생성

**도구:** Gemini (이미지 생성 모드)

### ✔️ 작업 내용
- 레퍼런스 이미지 확보 (Pinterest, Google 등 활용)
- 원하는 스타일 미리 수집

### ✔️ 프롬프트 작성법
- 단순 명령어 ❌  
- 구체적 설명 + 조건 조합 ⭕

**예시 프롬프트**

이 스타일로 백호 캐릭터 만들어 줘. 사람처럼 서서 팔짱 끼고 있는 포즈, 배경 없이 캐릭터만.

### ✔️ 디테일 추가
- 아이템 이미지 함께 제공
- 예시:

이 캐릭터에 이 선글라스 씌워 줘

---

## 📌 2단계: 시장 조사 및 영상 구조 분석

**도구:** YouTube, Gemini

### ✔️ 벤치마킹 채널 찾기
- 구독자: 1만 미만
- 조회수: 30만 이상 영상 중심 탐색

### ✔️ 구조 분석 요청

**분석 프롬프트**

이 영상 분석해서 구조랑 패턴 정리해 줘. 도입부, 전개, 클라이맥스, 마무리가 어떻게 구성되는지 알려줘.

---

## 📌 3단계: 스토리보드 및 장면 기획

**도구:** Gemini

### ✔️ 작업 내용
- 캐릭터 + 분석된 구조 결합
- 쇼츠용 기획안 제작

### ✔️ 기획 프롬프트 예시

이 캐릭터로 유튜브 쇼츠를 만들려고 해. 분석한 영상 구조를 참고해서 스토리랑 장면을 기획해 줘.

장면별 설정, 표정, 대사, 영상 길이까지 포함해 줘.

### ✔️ 결과물 포함 요소
- 장면 번호
- 배경
- 캐릭터 표정
- 대사
- 컷 길이

---

## 📌 4단계: 장면별 소스 이미지 생성

**도구:** Gemini (이미지 생성 모드)

### ✔️ 목적
- 비주얼 일관성 유지
- 영상 제작 전 이미지 기반 확보

### ✔️ 실행 방법
- 3단계에서 나온 장면별 프롬프트 사용
- 연속된 장면 이미지 생성

---

## 📌 5단계: 영상 애니메이팅

**도구:** NanoBanana (Google Flow)

### ✔️ 기본 설정
- Mode: `Frame Video`
- 테스트 모델 사용 (크레딧 절약)

### ✔️ 비율 설정
- 쇼츠용: `9:16` 세로 비율 필수

### ✔️ 작업 순서
1. 이미지 업로드
2. 움직임 프롬프트 입력
3. 영상 파일 생성

---

## 📌 6단계: 최종 편집 및 효과 추가

**도구:** CapCut (PC / 모바일)

### ✔️ 편집 단계
1. 영상 클립 타임라인 배치
2. BGM 삽입
3. 자막 추가
4. 효과 적용

### ✔️ 오디오
- [오디오] 탭 활용
- 분위기에 맞는 음악 선택

### ✔️ 자막
- [텍스트] 기능 활용
- 핵심 문장 강조

### ✔️ 내보내기
- 해상도: 1080p
- 플랫폼 업로드용 저장
